# 🥞 The Parotta Hub

A static web application for an authentic South Indian parotta stall based in Madurai, Tamil Nadu. Built as a learning project covering core HTML, CSS, and jQuery concepts.

## 🌐 Live Preview

Open `parotta.html` directly in any browser — no server or build step needed.

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (inline in `<style>`) |
| Interactivity | jQuery 3.7.1 (CDN) |
| Fonts | Google Fonts — Poppins, Merriweather |
| Map | OpenStreetMap embed (iframe) |

## 📚 Concepts Covered

### HTML
Document Structure, Elements & Tags, Attributes, Headings, Paragraphs, Text Formatting (`strong`, `em`, `mark`, `del`, `ins`, `sub`, `sup`), Hyperlinks, Images, Ordered/Unordered/Description Lists, Tables, Forms, Input Types (text, email, tel, number, date, time, range, file, url, radio, checkbox, select), Labels & Validation, Semantic HTML (`header`, `nav`, `main`, `section`, `article`, `aside`, `figure`, `figcaption`, `footer`, `address`, `time`, `details`), HTML5 Audio & Video, Iframes, Meta Tags, Accessibility (ARIA roles, aria-label, aria-labelledby, alt text), Entities & Special Characters, HTML Best Practices

### CSS
CSS Syntax, Selectors, Specificity, Colors & Backgrounds, Typography & Fonts, Units (px, %, rem, em, vh, vw), Box Model, Margin/Padding/Border, Display Property, Positioning (Static, Relative, Absolute, Fixed, Sticky), Overflow, Z-Index, Flexbox, CSS Grid, Responsive Design, Media Queries, Pseudo-classes (`:hover`, `:focus`, `:active`, `:valid`, `:invalid`, `:nth-child`), Pseudo-elements (`::before`, `::after`, `::first-line`), Transitions, Transformations, Animations (`@keyframes`), CSS Variables, Visibility & Opacity, Object Fit & Object Position, CSS Functions (`calc`, `min`, `max`, `clamp`)

### jQuery
jQuery Syntax, Selectors, DOM Manipulation, Event Handling, Traversing Elements, Show/Hide, Effects & Animations, Forms (`.serialize()`, `.serializeArray()`), DOM Creation & Removal, JSON Handling, AJAX Basics (`$.get`, `$.post`, `$.ajax`), Utility Functions (`$.each`, `$.map`, `$.grep`, `$.extend`), Method Chaining

## ✨ Features

- Sticky navigation with smooth scroll
- Animated hero section
- CSS Grid menu with 12 parotta varieties
- Dynamic Order Summary built with jQuery DOM creation
- Today's Specials section loaded from inline JSON
- Contact form with jQuery validation
- AJAX calls to JSONPlaceholder (demo)
- Scrollable image gallery
- FAQ section with slide toggle
- Fixed scroll-to-top button
- Toast notification system
- Responsive layout with media queries
